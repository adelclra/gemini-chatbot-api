# Produktif AI

Asisten produktivitas pribadi berbahasa Indonesia yang membantu kamu berhenti overthinking dan langsung eksekusi.

Produktif AI dirancang untuk pengguna yang ingin:

- menentukan prioritas tugas dengan cepat
- menjaga fokus harian tetap jelas
- menyusun rencana kerja yang realistis
- dapat dorongan motivasi saat mulai prokrastinasi

## Kenapa Produktif AI?

- Respons AI natural dalam Bahasa Indonesia
- Quick actions yang praktis: Tambah Tugas, Set Target, Buat Rencana
- UI modern dan ringan, nyaman dipakai harian
- Tugas dan target tersimpan otomatis di browser

## Cocok Untuk

- Mahasiswa yang ingin lebih konsisten mengerjakan tugas
- Freelancer yang butuh struktur kerja harian
- Profesional yang ingin menjaga fokus dan produktivitas
- Siapa pun yang ingin "lebih rapi" dalam mengelola to-do list

## Cara Coba Cepat

```bash
npm install
node index.js
```

Buka aplikasi di http://localhost:3000

Buat file `.env` di root project:

```env
GOOGLE_API_KEY=your_api_key_here
```

## Ringkasan Fitur

- Chat produktivitas berbasis Gemini API
- Daftar tugas aktif dengan akses cepat
- Target fokus harian
- Saran prompt siap pakai untuk memulai percakapan
- Rencana harian berdasarkan tugas yang kamu input

## Visi Produk

Produktif AI bukan sekadar chatbot tanya-jawab. Tujuannya adalah jadi partner kerja harian yang membantu kamu lebih fokus, lebih konsisten, dan lebih tenang saat menyelesaikan tugas.
